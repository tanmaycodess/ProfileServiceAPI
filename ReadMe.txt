# 🧑‍💼 Profile Service API

A microservice that handles authenticated user profile operations including profile updates, avatar uploads (Cloudinary/S3), and user preference management.

---

## 🔧 Tech Stack

- **Node.js** with **Express**
- **MongoDB** with **Mongoose**
- **passport** for authentication
- **Multer** for file upload
- **Cloudinary** or **AWS S3** for media storage
- **dotenv** for environment configuration

---

## 🚀 Features

- 🔐 **JWT-based Authentication** (Protected routes)
- 📝 **View / Update Profile** (`GET`, `PATCH`)
- 📸 **Upload Profile Picture** (Cloudinary/S3)
- 🎛️ **Save User Preferences** (e.g., theme, language)
